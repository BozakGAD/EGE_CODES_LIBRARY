# 9 Задание
## Пример взят с [имбового сайта](https://kompege.ru/) для подготовки к ЕГЭ.
Откройте файл электронной таблицы, содержащей в каждой строке семь целых чисел. Определить количество строк таблицы, для которых выполнены оба условия:
- в строке одно число повторяется трижды, остальные числа различны;
- максимальное число строки не повторяется.
В ответе запишите только число.
Файл к заданию: [9.ods](https://kompege.ru/files/gBssAOOdTBW.ods)
### Ответ: 1595
